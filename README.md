# Bravio Media

Neo-brutalist creative agency landing page built with React, TypeScript, and Vite.

## Stack
- React 19
- TypeScript
- Vite
- Plain CSS with design tokens

## Project structure
```text
src/
  components/
    layout/
    sections/
    ui/
  data/
    siteContent.ts
  styles/
    tokens.css
    base.css
    app.css
  App.tsx
  index.css
  main.tsx
```

## Run
```bash
pnpm install
pnpm dev
```

## Build and lint
```bash
pnpm build
pnpm lint
```

## Design notes
- Neo-brutalist system with thick borders, hard shadows, flat color surfaces, and oversized headlines
- Reusable data-driven sections for services, work, testimonials, process, and FAQ
- Semantic landmarks and accessible accordion/button states
